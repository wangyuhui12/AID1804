
shell命令

命令　[选项] [参数]

１、三个部分，每个部分之间一定要有空格
２、选项和参数可以没有，有时候选项和参数的位置比较灵活

例如：
cmd -r abc
cmd abc -r
cmd -r abc -f def

选项：辅助命令进行功能的丰富和细化
    选项的表达必须以　-　作为引导 ls -l
    如果有多个选项可以表示如下
    ls -l -a  <--->   ls -la

参数：命令要操作的目标，并不是所有的命令都可以加参数
*一般参数为一个路径或者精确到一个文件

rm  删除一个文件

rm [选项] 文件/文件夹
    -r　    删除一个文件夹及文件夹内所有内容
    -i      删除前提示
    -f      强制删除
删除非空文件夹：　rm -rf   文件夹

man     提供shell帮助

man 命令

cp　 复制文件或文件夹到某个位置
cp [选项] 文件/文件夹  目标位置

cp re python
1、如果第二个参数是一个存在路径文件夹，则表示将re拷贝到该位置
２、如果第二个参数是一个不存在的路径文件，则表示re另存为一个内容

-a　 如果要复制的内容是文件夹则使用该选项

mv 移动一个文件或文件夹

mv 文件/文件夹（可以为多个）    目标位置

参数和cp命令相同，移动文件夹也不需要选项
